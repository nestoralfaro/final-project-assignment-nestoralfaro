# OpenSpot (App)
Source code goes here. Available args:
- `--video [path_to_mp4]`: set source to video (mostly for testing).
- `--mpf`: set milliseconds per new frame read (if you need to avoid the overhead of livestreaming frames).
- `-c`: set source to camera.
- `-d`: convert to daemon.
- `-m`: enable email notifications (set up with `~/.msmtprc`)
